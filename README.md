Experience authentic artisanal pizza by chef. Cozy atmosphere, fresh ingredients, and unforgettable flavors. Book your table online and enjoy a unique dining experience!

## Features

-   Responsive design;
-   Food menu selection;
-   Navigation;
-   Home section;
-   Menu section;
-   About section;
-   Contact section.

## Technologies

-   Next.js;
-   Tailwind CSS;
-   React.js;
-   TypeScript;
-   Docker.
